FinRadar is an Android application designed to detect, categorize, and track recurring subscription payments by analyzing bank SMS and notifications locally on the device.

## Features
-   **Offline-First**: All data processing happens on the device.
-   **Smart Detection**: Automatically identifies subscriptions from transaction history.
-   **Price Alerts**: Warns you if a subscription price increases by more than 5%.
-   **Secure**: Uses local encryption and biometric authentication.

## Tech Stack
-   **Language**: Kotlin
-   **UI**: Jetpack Compose (Material 3)
-   **Architecture**: MVVM + Clean Architecture
-   **DI**: Hilt
-   **Database**: Room
-   **Async**: Coroutines + Flow

## Getting Started 
1.  Clone the repository.
2.  Open in Android Studio Iguana or later.
3.  Sync Gradle.
4.  Run on an emulator or device (API 26+).
